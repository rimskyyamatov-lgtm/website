# Blog Website

A simple HTML-based blog hosted on GitHub Pages.

## Features

- Clean and responsive design
- Easy to customize
- Hosted on GitHub Pages
- Built with HTML, CSS, and JavaScript

## How to Use

1. Edit `index.html` to add your blog posts
2. Customize `styles.css` for your desired look
3. Add interactivity with `script.js`
4. Push changes to the `main` branch
5. Your site will be available at: `https://rimskyyamatov-lgtm.github.io/website/`

## Blog Post Structure

To add a new blog post, add an `<article>` element:

```html
<article>
    <h2>Your Post Title</h2>
    <p>Posted on <time datetime="YYYY-MM-DD">Date</time></p>
    <p>Your post summary here. <a href="#">Read more</a></p>
</article>
```

## Customization

- Change colors in `styles.css`
- Modify the header/footer in `index.html`
- Add JavaScript functionality in `script.js`

## GitHub Pages Setup

1. Go to Settings → Pages
2. Select `main` branch as the source
3. Your site will be published at `https://rimskyyamatov-lgtm.github.io/website/`
